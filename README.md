## üöÄ Quick Start

### Backend Setup
```bash
cd ai4kg/backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend Setup
```bash
cd ai4kg/frontend
npm install
npm run dev
```

## üìö Documentation Development

The documentation is automatically deployed to GitHub Pages when changes are pushed to the `docs/` directory.

### Local Preview
```bash
cd docs
python -m http.server 3000
# Or use docsify
npm i docsify-cli -g
docsify serve .
```
